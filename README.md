## Keybinds

| Key              | Description           |
|------------------|-----------------------|
| <kbd>↑/↓</kbd>   | Move cursor           |
| <kbd>Enter</kbd> | Proceed / Open        |
| <kbd>Space</kbd> | Toggle multiselect    |
| <kbd>1..9</kbd>  | Jump to index in list |
| <kbd>Esc</kbd>   | Exit app              |
